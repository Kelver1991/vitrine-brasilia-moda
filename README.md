# Colcci Brasilia - conceito de site

Projeto inicial em HTML, CSS e JavaScript puro para estudar uma vitrine moderna inspirada no tipo de experiencia do ecommerce oficial da Colcci.

> Atencao: use nome, imagens, produtos, campanhas e identidade da Colcci somente se voce tiver autorizacao da marca, franquia ou responsavel comercial. Para portfolio, trate como projeto conceitual.

## Como abrir o site

1. Abra a pasta do projeto.
2. Clique duas vezes em `index.html`.
3. Edite textos, endereco, telefone e produtos nos arquivos `index.html` e `script.js`.

## Passo a passo do zero

1. Crie uma pasta para o projeto.
2. Dentro dela, crie os arquivos `index.html`, `styles.css` e `script.js`.
3. Monte primeiro a estrutura do HTML: cabecalho, hero, beneficios, colecoes, vitrine, loja e rodape.
4. Estilize o visual no CSS: cores, espacamentos, responsividade e imagens.
5. Use JavaScript para pequenas interacoes: filtro de produtos, formulario e futuros botoes de carrinho.
6. Teste no celular e no computador.
7. Troque imagens temporarias por imagens autorizadas.
8. Troque o link do WhatsApp em `index.html` pelo numero correto.
9. Publique no GitHub.
10. Depois, publique no GitHub Pages, Vercel ou Netlify.

## Como enviar para o GitHub

No terminal, dentro desta pasta:

```bash
git status
git add .
git commit -m "Cria site conceito Colcci Brasilia"
git branch -M main
git remote add origin https://github.com/SEU-USUARIO/NOME-DO-REPOSITORIO.git
git push -u origin main
```

Se o repositorio remoto ja existir configurado, use:

```bash
git remote -v
git push -u origin main
```
